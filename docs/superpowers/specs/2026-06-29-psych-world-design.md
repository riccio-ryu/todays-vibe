# 오늘난 — 심리 테스트 세계 설계 스펙

> **서비스명:** 오늘난  
> **작성일:** 2026-06-29  
> **상태:** 사용자 검토 대기

---

## 1. 개요

오늘운(밤/달 테마, 운세 33종)의 자매 세계.  
낮/해 테마의 심리 테스트 플랫폼으로, 메인 화면 스와이프로 전환.

| 구분 | 오늘운 | 오늘난 |
|------|--------|--------|
| 테마 | 밤 · 달 · 신비 | 낮 · 해 · 명료 |
| 배경 | `#0a0312` 딥 퍼플-블랙 | `#ebf5ff` Sky Wash |
| 목적 | 운세 · 점술 | 심리 진단 · 자기탐색 |
| AI | 사주·타로 해석 | 성격·감정·관계 분석 |

---

## 2. 메인 진입 구조

### 2-1. 스와이프 전환 (홈 화면)

```
← 스와이프                    스와이프 →
[오늘운 — 밤 테마]  ↔  [오늘맘 — 낮 테마]
```

- 전환 시 배경색·카드·폰트 컬러 전체 교체
- 전환 애니메이션: 수평 슬라이드 (Framer Motion)
- 메인 화면에서만 스와이프 가능 (개별 테스트 페이지는 해당 없음)
- 마지막 선택 세계를 localStorage에 기억

### 2-2. 심리 테스트 메인 레이아웃

```
┌─────────────────────────────────────────┐
│  [ 심리 진단 ]     심리 스낵            │  ← 토글 (pill 형태)
├─────────────────────────────────────────┤
│                                         │
│  심리 진단 탭: 카드 24개 그리드         │
│  심리 스낵 탭: 카드 18개 + 오늘의 스낵  │
│                                         │
└─────────────────────────────────────────┘
```

Geniestudio 레퍼런스 기반 낮 테마 적용.

| 역할 | 이름 | 값 |
|------|------|----|
| 페이지 배경 | Sky Wash | `#ebf5ff` |
| 카드 배경 | Paper White | `#fafdff` |
| 중첩 표면 | Pure White | `#ffffff` |
| 기본 텍스트 | Midnight Ink | `#0a0d12` |
| 보조 텍스트 | Stone | `#535862` |
| 흐린 텍스트 | Fog | `#93979f` |
| 주요 버튼 배경 | Pressed Charcoal | `#181d27` |
| 장식 강조 1 | Cornflower | `#4fbeff` |
| 장식 강조 2 | Amethyst | `#9552e0` |
| 장식 강조 3 | Tangerine | `#f26110` |
| 장식 강조 4 | Mustard | `#bb9915` |
| 진단형 칩 배경 | Morning Tint | `#cce7ff` |
| 스낵형 칩 배경 | Lilac Mist | `#f1e6ff` |

> 강조색 4종(Cornflower·Amethyst·Tangerine·Mustard)은 **장식·아이콘 전용**.  
> 버튼·텍스트·UI 상태에 절대 사용 금지.

### 타이포그래피

| 역할 | 폰트 | 굵기 | 크기 |
|------|------|------|------|
| 헤딩 | Aeonik (대체: Inter, Satoshi) | 500 | 20–48px |
| 본문/UI | Geist (대체: Inter) | 500/600 | 10–18px |

- 자간: 헤딩 `-0.02em`, 본문 `-0.01em`
- **이모지 사용 금지** — 아이콘은 라인 SVG 아이콘만 사용

### 형태 & 간격

| 요소 | 값 |
|------|----|
| 카드 반경 | 32px |
| 버튼 반경 | 9999px (pill) |
| 아이콘 반경 | 16px |
| 카드 패딩 | 24–40px |
| 섹션 간격 | 64–80px |
| 그림자 | `rgba(4,69,144,0.08) 0px 14px 20px 4px` |
| 테두리 | `1px solid #535862` |

### 진단형 카드 vs 스낵형 카드

**진단형** — Paper White + 1px Stone 테두리, 32px 반경, 40px 패딩  
제목(20px) · 설명 2줄 · 문항 수/소요 시간 · 척도 출처 · CTA 버튼

**스낵형** — Morning Tint 또는 Lilac Mist 배경, 테두리 없음, 32px 반경, 24px 패딩  
제목(18px) · 한 줄 설명 · 소요 시간 · CTA 버튼

## 4. 테스트 카탈로그 구조

### 4-1. 탭 구성

| 탭 | 콘텐츠 | 카드 수 (MVP) |
|----|--------|--------------|
| 심리 진단 | 검증된 척도 기반 테스트 | 12개 |
| 심리 스낵 | 재미·바이럴 테스트 | 10개 + 오늘의 스낵 위젯 |

### 4-2. 심리 진단 탭 MVP 12개

| # | 테스트 | AI | 근거 |
|---|--------|-----|------|
| 1 | MBTI 유형 진단 | 비AI | MBTI 척도 |
| 2 | 에니어그램 | 비AI | 에니어그램 척도 |
| 3 | 애착 유형 진단 | 비AI | ECR 척도 |
| 4 | 사랑의 언어 | 비AI | 5 Love Languages |
| 5 | 번아웃 위험도 | 비AI | MBI 척도 |
| 6 | 스트레스 지수 | 비AI | PSS-10 |
| 7 | 자존감 진단 | 비AI | Rosenberg 척도 |
| 8 | 공감 능력 (EQ) | 비AI | EQ 척도 |
| 9 | AI 성격 분석 | AI | 자유 입력 → Claude |
| 10 | AI 고민 분석 | AI | 자유 입력 → Claude |
| 11 | AI 감정 일기 | AI | 자유 입력 → Claude |
| 12 | AI 연애 상담 | AI | 자유 입력 → Claude |

> MBTI 10가지 변형(연애 MBTI, 직장 MBTI 등)은 **MBTI 카드 내부 모드**로 처리.  
> 별도 카드로 노출하지 않음.

### 4-3. 심리 스낵 탭 MVP 10개

| # | 테스트 | 특이점 |
|---|--------|--------|
| 1 | 동물 유형 | 결과 이미지 카드 공유 |
| 2 | 조선시대 직업 | 유머 결과, 높은 바이럴 |
| 3 | 나의 스낵 유형 | 과자 비유, 극한 공유율 |
| 4 | 꽃 유형 | 결과 카드 비주얼 핵심 |
| 5 | 연애 스타일 | 5가지 유형 |
| 6 | 사막을 건너는 동물 순서 | 가치관 반전 결과 |
| 7 | 친구들이 보는 나 | 친구 공유 CTA |
| 8 | 소비 성향 | 유머 결과 |
| 9 | 색깔 심리 | 순수 시각 인터랙션 |
| 10 | 초능력 테스트 | 공유 이미지 최적화 |

### 4-4. 오늘의 스낵 위젯

스낵 탭 상단 고정. 매일 1개 로테이션.  
붕어빵·지하철 빈자리·엘리베이터 행동·호텔 조식 등 마이크로 테스트.  
결과가 단 하나의 선택으로 즉시 나옴.

### 4-5. 카탈로그에서 제거한 항목

| 항목 | 이유 |
|------|------|
| 집착도 + 연락 집착도 (중복) | 하나로 통합 |
| AI 특화 15종 전체 노출 | 6개로 압축 (나머지는 카드 내 옵션) |
| 좌뇌/우뇌 테스트 | 진부하고 과학적 근거 약함 |
| AI 미래 성향 예측 | 결과 신뢰도 낮음, 모호 |
| MBTI 10종 별도 카드 | MBTI 카드 내 모드 선택으로 통합 |
| 붕어빵·지하철·엘리베이터 (단독) | 오늘의 스낵 위젯으로 통합 |

## 5. 테스트 진행 UX

### 5-1. 진행 화면 (질문 한 장씩)

```
┌───────────────────────────────────────┐
│  ← 뒤로      MBTI 유형 진단      ✕   │
│  ━━━━━━━━━━░░░░░░░░░░░░░░  5 / 20    │
│                                       │
│                                       │
│  갑작스럽게 주말 일정이 취소됐다.     │
│  당신은 가장 먼저 무엇을 하나요?      │
│                                       │
│  ┌───────────────────────────────┐   │
│  │  혼자 집에서 쉰다             │   │
│  └───────────────────────────────┘   │
│  ┌───────────────────────────────┐   │
│  │  친구에게 바로 연락한다       │   │
│  └───────────────────────────────┘   │
│  ┌───────────────────────────────┐   │
│  │  즉흥적으로 어딘가 나간다     │   │
│  └───────────────────────────────┘   │
│  ┌───────────────────────────────┐   │
│  │  다른 계획을 세운다           │   │
│  └───────────────────────────────┘   │
│                                       │
└───────────────────────────────────────┘
```

**규칙:**
- 선택지 탭 즉시 다음 문항으로 자동 전환 (제출 버튼 없음)
- 전환 애니메이션: 좌측으로 슬라이드 아웃 → 우측에서 슬라이드 인
- 상단 뒤로 버튼으로 이전 문항 수정 가능
- 진행률 바: 완료 문항 수 / 전체 문항 수 텍스트 병기
- 중간 이탈 시 진행 상태 localStorage 임시 저장 (재진입 시 이어하기)

### 5-2. AI 입력형 테스트 진행

```
┌───────────────────────────────────────┐
│  AI 고민 분석                    ✕   │
│                                       │
│  지금 가장 마음에 걸리는              │
│  고민을 자유롭게 적어주세요.          │
│                                       │
│  ┌───────────────────────────────┐   │
│  │                               │   │
│  │  (자유 입력)                  │   │
│  │                               │   │
│  └───────────────────────────────┘   │
│                                       │
│            [분석 시작하기]            │
└───────────────────────────────────────┘
```

- 최소 30자 이상 입력 유도 (글자 수 표시)
- 분석 중: 텍스트 스트리밍 로딩 인디케이터
- 기존 `FortuneResult` 스트리밍 컴포넌트 재사용

## 6. 결과 페이지

### 6-1. 진단형 결과 (4섹션)

```
━━━━━━ 섹션 1: 유형 요약 ━━━━━━
당신의 유형은
INFJ — 옹호자
"이상과 현실 사이에서 다리를 놓는 조용한 완벽주의자"

━━━━━━ 섹션 2: 축 분석 ━━━━━━
E ░░░░░░░░████████████ I  72%
S ░░░░░░░░░░░░░██████ N  38%
T ░░░░░░░░███████████ F  68%
P ░░░░░░░░░░░░░██████ J  34%

━━━━━━ 섹션 3: 상세 해석 ━━━━━━
강점 / 주의점 / 관계 스타일
(300–500자 텍스트)

━━━━━━ 섹션 4: 액션 ━━━━━━
[결과 저장]  [공유하기]
연관 테스트: 애착 유형 · 번아웃 위험도
```

**신뢰도 장치:**
- 척도 출처는 **테스트 시작 전 카드 설명**에 표시 ("MBTI 척도 기반 · 20문항")
- 결과 페이지 하단에 한 줄 면책: "이 결과는 참고용이며 전문 심리 진단을 대체하지 않습니다"

### 6-2. 스낵형 결과 (1섹션)

```
당신은 고양이형
독립적이고 자기 페이스를 지키는 당신.
깊이 있는 관계를 소중히 하지만
혼자만의 시간도 꼭 필요합니다.

[공유하기]  [다시 하기]
```

- Morning Tint 또는 Lilac Mist 배경
- 아이콘: 라인 SVG만 (이모지 없음)
- 공유 버튼 최우선 배치

## 7. 데이터 구조

### Firestore 컬렉션

```javascript
// psych_readings/{readingId}
{
  type: 'diagnostic' | 'snack' | 'ai',
  slug: 'mbti' | 'burnout' | 'animal' | ...,
  userId: string,       // optional (비로그인 가능)
  input: {
    answers?: number[], // 비AI: 선택지 인덱스 배열
    text?: string,      // AI: 자유 입력 텍스트
  },
  result: {
    typeId?: string,    // 유형 코드 (예: 'INFJ')
    scores?: Record<string, number>,  // 축별 점수
    aiText?: string,    // AI 스트리밍 결과
  },
  createdAt: Timestamp,
  isPublic: boolean,
}
```

## 9. 구현 순서 (제안)

### Phase 1 — 기반 (비AI 진단형 3개)
1. `(psych)` 라우트 그룹 + 낮 테마 레이아웃
2. 공통 `<QuizLayout />` 컴포넌트 (질문 한 장씩 + 진행률 바)
3. MBTI 데이터 JSON + 스코어링 엔진
4. 공통 `<DiagnosticResult />` 결과 컴포넌트 (축 차트 포함)
5. 번아웃 · 자존감 추가 (JSON만 교체)

### Phase 2 — 스낵형 + 메인 토글
6. `<SnackResult />` 결과 컴포넌트
7. 동물 유형 · 조선시대 직업 · 스낵 유형 JSON
8. 심리 테스트 메인 페이지 (진단/스낵 토글 + 오늘의 스낵 위젯)
9. 홈 화면 스와이프 전환 (오늘운 ↔ 오늘난)

### Phase 3 — AI 연동
10. AI 입력형 테스트 UI (자유 입력 + 스트리밍)
11. AI 고민 분석 · 감정 일기 · 연애 상담 · 성격 분석 4종

### Phase 4 — 완성도
12. 결과 공유 이미지 생성
13. 즐겨찾기 (오늘운과 동일 구조)
14. 결과 저장 → 마이페이지 연동
