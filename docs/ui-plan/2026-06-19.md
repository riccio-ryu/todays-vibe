상승 표면:   #10093a  (Deep Indigo)

타이포:      weight 500 (bold 사용 안 함), 제목·본문 모두 medium

## 현재 todays-vibe UI 진단

### 잘 되어 있는 것
- `TimeBackground` — 시간대별 HSL 보간 배경은 독창적 강점, 유지 권장
- `SkyLayer` — 별 + 구름 레이어 이미 구현됨
- 카드 컴포넌트 재사용 구조
- 신비로운 분위기 기본 방향 OK

### 핵심 문제
> 색상이 너무 많고, "보라 우주"라는 단일 테마가 흐트러져 있음

- 액센트가 6종 이상 혼재: `purple-600`, `amber-300`, `yellow-300`, `emerald-300`, `blue-300` ...
- `text-white` (순수 흰색) 사용으로 "차가운 디지털" 느낌
- 카드 배경 그라데이션이 복잡하여 시선 분산
- `font-bold` 남용으로 전체적으로 무거운 인상

## 개선 항목 상세

---

### 1. 텍스트 색상 라벤더화 ⭐ 즉시 효과 큼

| 현재 | 변경 후 |
|------|---------|
| `text-white` (`#ffffff`) | `text-[#f4f0ff]` |
| `text-white/40`, `text-white/50`, `text-white/60` | `text-[#a8a6b7]` (불투명 회보라) |
| `text-purple-300/80`, `text-purple-400/70` | `text-[#a8a6b7]` 으로 통일 |
| `text-purple-100`, `text-purple-200` | `text-[#f4f0ff]` |

**적용 파일**: OracleHeader, HeroCard, Header, FortuneGrid 카드 컴포넌트

**효과**: 순수 흰색 제거만으로도 전체 팔레트가 "보라 우주" 톤으로 통일됨

---

### 2. 액센트 단일화 (`#9382ff`) ⭐ 브랜드 정체성 핵심

**현재 문제**:
```
AI 뱃지     → amber-300
점수 (high) → yellow-300
점수 (mid)  → emerald-300
점수 (low)  → blue-300
키워드 칩   → purple-600/20
버튼 강조   → purple-600
```

**변경 후** — 모두 `#9382ff` 단일 액센트로:
```
AI 뱃지     → bg-[#9382ff]/20 text-[#9382ff]
점수        → text-[#9382ff] (고/중/저 구분 제거)
키워드 칩   → border-[#9382ff]/30 text-[#9382ff]
버튼 강조   → bg-[#9382ff] (hover: bg-[#7a6be0])
```

**예외 허용**: 에러 → red, 성공 → 단발성만

---

### 3. 코스믹 그라데이션 텍스트 ⭐ 브랜드 임팩트

Reflect Notes의 핵심 시그니처: `#e59cff → #ba9cff → #9cb2ff`

**적용 위치**:

```tsx
// globals.css 추가
.cosmic-text {
  background: linear-gradient(90.01deg, #e59cff 0.01%, #ba9cff 50.01%, #9cb2ff 100%);
  -webkit-background-clip: text;
  -webkit-text-fill-color: transparent;
  background-clip: text;
}
```

적용:
- `OracleHeader` — `✦ Today's Vibe ✦` 배지
- `HeroCard` — "오늘의 기운이 도착했어요" 타이틀
- AI 뱃지 텍스트
- 섹션 제목 (인기 운세, 빠른 메뉴 등)

---

### 4. 카드 스타일 — 인셋 림라이트 글로우

**현재**:
```css
bg-gradient-to-br from-purple-900/40 via-indigo-900/25 to-blue-900/20
border border-white/10
```

**변경 후**:
```css
background: #060317;  /* 또는 tailwind: bg-[#060317] */
box-shadow: rgba(255,255,255,0.04) 0px 0px 24px 0px inset;
/* border 제거 */
```

tailwind 커스텀:
```tsx
// tailwind.config.ts 또는 globals.css에 추가
.card-glow {
  background: #060317;
  box-shadow: rgba(255,255,255,0.04) 0px 0px 24px 0px inset;
}

// hover시 액센트 글로우
.card-glow:hover {
  box-shadow:
    rgba(255,255,255,0.04) 0px 0px 24px 0px inset,
    rgba(147,130,255,0.08) 0px 0px 32px 0px;
}
```

**효과**: 카드마다 달랐던 그라데이션 통일 → 깊이감은 살리면서 산만함 제거

---

### 5. 타이포그래피 — font-bold → font-medium

| 현재 | 변경 후 |
|------|---------|
| `font-bold text-white text-xl` | `font-medium text-[#f4f0ff] text-xl` |
| `font-semibold` | `font-medium` |
| `font-bold` 헤딩 전체 | `font-medium` |

`font-bold` 허용 예외: 점수 숫자, 강조 수치만

---

### 6. Floating Pill 헤더

**현재**: 풀 너비 고정 헤더 (`fixed top-0 w-full ... border-b border-white/10`)

**변경 후**:
```tsx
<header className="
  fixed top-4 left-1/2 -translate-x-1/2 z-50
  flex items-center justify-between
  px-5 py-2.5
  rounded-full
  bg-[#060317]/80 backdrop-blur-xl
  shadow-[rgba(255,255,255,0.04)_0px_0px_24px_0px_inset,rgba(147,130,255,0.1)_0px_0px_32px_0px]
  w-[calc(100%-2rem)] max-w-2xl
">
```

**장점**: 배경이 보여 시각적 깊이감, 모바일에서도 세련된 인상
**단점**: 콘텐츠 최상단이 pill에 가려질 수 있어 `pt` 조정 필요 (`pt-[72px]` → `pt-[80px]`)

---

### 7. 버튼 radius 조정

| 위치 | 현재 | 변경 후 |
|------|------|---------|
| CTA 주요 버튼 | `rounded-full` (pill) | 유지 ✅ |
| 로그인/회원가입 | `rounded-lg` (8px) | `rounded-[5px]` |
| 드롭다운 메뉴 | `rounded-xl` (12px) | `rounded-[8px]` |
| 뱃지/칩 | `rounded-full` (pill) | 유지 ✅ |

---

### 8. 스타 필드 CSS box-shadow 방식 (선택)

현재 `SkyLayer`는 별 80개를 DOM 요소로 렌더링. CSS box-shadow 방식이 성능상 유리:

```css
/* globals.css */
.star-field::after {
  content: "";
  position: fixed;
  inset: 0;
  pointer-events: none;
  /* 40개 tiny dots — 1px white/8 */
  box-shadow:
    120px 80px 0 rgba(255,255,255,0.08),
    340px 210px 0 rgba(255,255,255,0.06),
    /* ... 38개 더 ... */
  ;
}
```

현재 SkyLayer가 이미 잘 작동하므로 필수 변경은 아님, 선택 사항.
